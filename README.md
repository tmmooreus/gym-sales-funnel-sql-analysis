# GTM Sales Funnel SQL Analysis

## Overview
This project analyzes CRM style GTM and sales funnel data using SQL.

The project includes relational tables for:
- Leads
- Deals

Using SQL queries, the project analyzes:
- Lead source performance
- Revenue by pipeline stage
- Average deal size
- Pipeline distribution
- Unconverted leads
- Revenue attribution by lead source

---

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- COUNT
- SUM
- AVG
- INNER JOIN
- LEFT JOIN
- Table aliases

---

## Example Business Questions Answered
- Which lead sources generate the most revenue?
- Which pipeline stages contain the most revenue?
- Which leads failed to convert into deals?
- What is the average deal size by stage?
- How many deals exist in each pipeline stage?
